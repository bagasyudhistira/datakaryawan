1. Install XAMPP
2. Buka XAMPP
3. Tekan tombol "Start" pada Apache dan MySQL
4. Pindahkan direktori ke <tempat install XAMPP>/htdocs/
5. Buka browser
6. Ketik "localhost/phpmyadmin"
7. Buat sebuah database baru dengan nama "datakaryawan"
8. Buka database "datakaryawan"
9. Pilih tab "Import"
10. Tekan tombol "Choose File" kemudian memilih file "datakaryawan.sql"
11. Uncheck Partial Import dan "Do not use AUTO_INCREMENT for zero values"
12. Tekan tombol "Go"
13. Buat tab baru dan ketik "localhost/datakaryawan"
